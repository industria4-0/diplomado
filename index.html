DIPLOMADO EN INDUSTRIA 4.0 APLICADA A LA AUTOMATIZACIÓN

Tecnologías 4.0 en la Automatización

Descripción del diplomado

Objetivo del Diplomado

Desarrollar habilidades digitales en tecnologías de la Industria 4.0 para proponer soluciones en proyectos de Automatización Industrial.

Horarios del Diplomado

Modalidad: 100% Remota y sincrona. Sesiones grabadas y compartidas.

Horarios: Jueves y Viernes con sesiones de 6 PM a 10 PM.

Cantidad Módulos: 3 módulos/ Cada módulo de 8 sesiones.

Comienzo Módulo 1: Jueves 1 de Septiembre de 2022.

Comienzo Módulo 2: Jueves 6 de Octubre de 2022.

Comienzo Módulo 3: Jueves 3 de Noviembre de 2022.

¿Porque estudiar este Diplomado?

Primero: Es 100% remoto sincrono con acompañamiento de docente, todas las clases se graban y se comparten con el estudiante.

Segundo: Nuestro valor agregado es que es un Diplomado dinamico, con contenidos de uso de Tecnologías 4.0 como lo son Gemelo Digital, Aplicaciones moviles, Computación en la nube e Internet de las cosas

Tercero: Se aprende el uso de Tecnologias 3.0 como lo son programacion de PLCs de la marca Allen Bradley y Configuracion de sistemas SCADAs.

Módulos del Diplomado

PLCs y Gemelo Digital

Aprender a programar PLCs y probar estas lógicas en un gemelo digital

Aplicaciones móviles con funciones de SCADA

Configurar una aplicación móvil para que cumpla funciones de un sistema SCADA.

Bots e Internet de las Cosas en la Automatización

Crear y configurar un Bot para que a través de Internet de las Cosas se comunique con un PLC.

Módulo 1: PLCs Y GEMELO DIGITAL.

En este módulo se aprende a programar PLCs de la marca Allen Bradley usando lenguaje Ladder y tambien se aprende a usar el sofware Factory IO para probar estas logicas de PLC.

Módulo 2: APLICACIONES MÓVILES CON FUNCIONES DE SCADA.

En este módulo de aprende a usar el SCADA de marca Ignition que posee la funcion de usar una aplicación móvil para supervisar y operar logicas de PLC.

Módulo 3: BOTS E IOT EN LA AUTOMATIZACIÓN.

En este módulo se aprende a crear un Bot en Telegram ademas de entrenarlo a traves de IBM Watson Assitant y conectarlo a un PLC usando el software Node-RED.

Retos 2022-2

A continuacion se presenta la descripcion de los retos propuestos para el Diplomado Versión 2022-2.

Reto PLC y Gemelo Digital

Diseñar una lógica en Ladder para la operación de un proceso industrial teniendo en cuenta lo siguiente:

• Programar la lógica en el PLC usando lenguaje Ladder.
• Probar lógica de PLC en un Gemelo Digital.

Reto Aplicaciones Móviles con funciones de SCADA

Configurar una aplicacion movil para usarla como SCADA para la operación de un proceso indsutrial teniendo en cuenta lo siguiente:

• Programar la lógica en el PLC usando lenguaje Ladder.
• Probar lógica de PLC en un Gemelo Digital.
• Usar una aplicacion móvil con funciones de SCADA para la operacion de un proceso industrial.

Reto Bots e IoT en la Automatización.

Crear un Bot entrenado con IBM Watson Assistant para la operación de un proceso indsutrial teniendo en cuenta lo siguiente:

• Programar la lógica en el PLC usando lenguaje Ladder.
• Probar lógica de PLC en un Gemelo Digital.
• Usar un bot para la operacion de un proceso industrial.
• Usar un dashboard en un servicio cloud para la supervisión y oeración de un proceso industrial.
